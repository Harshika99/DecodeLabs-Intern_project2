Student Grade Calculator

A simple Java console-based application that calculates a student’s total marks, average percentage, and grade based on marks entered for multiple subjects.

**Features**:
Takes marks input for multiple subjects
Calculates total marks
Calculates average percentage
Assigns grades based on percentage
Displays final result clearly
Validates marks input

**Concepts Used**:
Java Basics
Loops
Conditional Statements
Arrays
Methods
User Input using Scanner

**Technologies**:
Java
VS Code 

**How It Works**:
User enters the number of subjects
User enters marks for each subject
Program calculates:
Total Marks
Percentage
Grade
Final result is displayed on the console
